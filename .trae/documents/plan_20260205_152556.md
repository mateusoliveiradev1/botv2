O AutoMod atual apenas deleta a mensagem e avisa, mas não tem "memória" nem penalidades crescentes. Para resolver isso, vou implementar um sistema de **Advertências (Warns) Progressivas**.

**O Plano:**

1.  **Gerenciador de Advertências (`WarningManager`)**:
    *   Criar um sistema para salvar advertências em `data/warnings.json`.
    *   Cada vez que o AutoMod agir, o usuário receberá 1 advertência.

2.  **Escada de Punições (Configurável)**:
    *   **3 Advertências**: Timeout de **10 minutos** (Mute).
    *   **5 Advertências**: Timeout de **1 hora**.
    *   **7 Advertências**: Kick (Expulsão) ou Timeout de **24 horas** (vou configurar Timeout de 24h por padrão para ser menos destrutivo, mas posso mudar para Kick se preferir).

3.  **Integração**:
    *   O `AutoMod` chamará esse gerenciador automaticamente.
    *   O usuário será notificado no chat: "Você recebeu uma advertência (Total: X). Punição aplicada: Y".

4.  **Comandos de Admin (Opcional, mas recomendado)**:
    *   Se quiser, posso adicionar comandos como `!warns @usuario` para ver o histórico e `!unwarn @usuario` para remover, mas o foco agora é a automação.
