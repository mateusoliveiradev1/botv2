Entendi os dois problemas. Vamos corrigir:

1.  **Bug do "Desconhecido" no Link**:
    *   Isso acontece porque o Discord às vezes se perde com links para canais que você *já* está visualizando ou se o cache do bot não estiver 100% atualizado com o nome do canal.
    *   **Correção**: Vou mudar a lógica. Em vez de tentar gerar um link complexo, se o usuário clicar em "Suporte Humano", o bot vai enviar uma mensagem efêmera (invisível para os outros) dizendo: "Clique aqui para ir ao canal: <#ID_DO_CANAL>". O Discord renderiza `<#ID>` como um link clicável perfeito que funciona em qualquer dispositivo.

2.  **Botão "Reportar Bug" Desativado**:
    *   O botão estava com `.setDisabled(true)` no código (provavelmente deixado como placeholder).
    *   **Correção**: Vou ativar o botão (`.setDisabled(false)`) e criar um handler para ele. Quando clicar, vai abrir um Modal simples pedindo a descrição do bug.

Vou implementar essas correções no `ajuda.ts` e no `interactionCreate.ts`.
