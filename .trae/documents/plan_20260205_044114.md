# Atualização do Sistema de Ranking Automático

## 1. Modificar Gerenciador de Setup (`src/modules/setup/manager.ts`)
Implementar a nova lógica de criação de canais de ranking e webhooks.

### Novo Método: `createRankingSystem()`
- **Categoria:** Cria '📊 RANKINGS' se não existir.
- **Canais:** Cria os 4 canais solicitados:
  - `📅-ranking-semanal`
  - `📆-ranking-mensal`
  - `🏆-ranking-competitivo`
  - `⚔️-ranking-clas`
- **Webhooks:**
  - Cria um Webhook "PUBG Tracker Rankings" em cada canal.
  - Usa o avatar do próprio bot para o Webhook.
  - **Exibição:** Imprime as URLs dos Webhooks no terminal (Logger) para você configurar no Lovable.

## 2. Remover Comando Manual
- Deletar o arquivo `src/commands/data/ranking.ts` para evitar conflito com o novo sistema automático.

## Resultado Esperado
Ao rodar `/setup`:
1. A nova categoria e canais aparecerão.
2. No terminal do bot, você verá uma lista de URLs de Webhooks (uma para cada canal) para copiar e colar na configuração do Lovable.
3. O comando `/ranking` deixará de existir.
